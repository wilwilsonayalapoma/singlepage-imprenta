# Imprenta Nuevo Mundo — Single Page (React)

Pequeña SPA en React con header responsive, carrusel automático en la sección de productos y footer.

Instrucciones rápidas:

1. Instalar dependencias:

```bash
npm install
```

2. Iniciar servidor de desarrollo:

```bash
npm start
```

3. Reemplaza las imágenes del carrusel en `src/components/ProductsCarousel.js` por tus propias imágenes o agrega assets en `public/`.

Estructura creada:
- `public/` - plantilla HTML
- `src/components/` - `Header`, `ProductsCarousel`, `Footer`
- `src/` - `App.js`, `index.js`, `index.css`

Notas:
- Este proyecto usa webpack y Babel para ser ligero y no requerir create-react-app. Si prefieres CRA o Vite puedo migrarlo.
- Cambia los textos y logos SVG por tus recursos reales.
